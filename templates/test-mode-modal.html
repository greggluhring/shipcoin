Shipcoin is in test mode

Shipcoin only returns sample USPS test labels. You don't have to pay for these labels, but can't use them to actually ship packages.

It's not possible to purchase paid shipping labels on Shipcoin.

To get paid postage, we recommend any of the following options:

  • If you're a developer, you can host your own instance of Shipcoin by downloading the open-source code at https://github.com/oneminutepostage/shipcoin and running it with a production Shippo API token.
  • Directly use Shippo's web interface to pay with credit card.
  • Use Shipnik to pay with Bitcoin.
